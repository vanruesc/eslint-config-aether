# Eslint Config Delta

A custom code style for [ESLint](https://eslint.org/).


## Installation

```sh
npm install eslint-config-delta --save-dev
``` 


## Usage

Choose one of the following options to load the configuration.

#### package.json

```js
{
	"eslintConfig": {
		"extends": "delta"
	}
}
```

#### .eslintrc

```js
{
	"extends": "delta"
}
```
